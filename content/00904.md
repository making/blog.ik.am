[ds4(DwarfStar 4)](https://github.com/antirez/ds4)がDGX Sparkに対応したので、試してみます。

```bash
sudo mkdir -p /opt/ds4
sudo chown -R $USER:$USER /opt/ds4
```

```bash
git clone https://github.com/antirez/ds4.git .
```

```bash
$ git rev-parse --short HEAD
920f987
```

```bash
./download_model.sh q2 
```

```bash
make
```

```bash
./ds4-server --ctx 100000 --kv-disk-dir /tmp/ds4-kv --kv-disk-space-mb 8192 --host 0.0.0.0
```
