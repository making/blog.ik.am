[giscus](https://giscus.app) によるコメント機能を各ブログ記事に追加してみました。

↓ の図のように、各記事の下部にコメントフォームが表示されます。  

![comment-form.png](https://s3.ik.am/ikam/_/1769049744006_comment-form.png)

コメントを書くには GitHub アカウントでサインインする必要があります。投稿されたコメントは、当ブログの GitHub リポジトリ内の Discussions に保存され、オープンソースの形で公開されます。

ぜひ、感想や質問などをご自由にコメントしてください。
