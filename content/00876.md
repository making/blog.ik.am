LLM向けに、全記事の一覧とメタ情報をまとめた **[llms.txt](/llms.txt)** を用意しました。  
このファイルには、エントリーの ID、タイトル、タグ、作成日時などがタブ区切りで記載されており、検索やバッチ処理に便利です。

さらに、各記事の URL の末尾に **`.md`** を付加すると、HTML ではなく **Markdown 形式** の本文を直接取得できるようにしました。  
（例）  
- 通常の閲覧 URL: `https://ik.am/entries/876`  
- Markdown 取得 URL: `https://ik.am/entries/876.md`

この `.md` エンドポイントは、記事の構造化データやテキスト解析を行う際にそのまま利用でき、余計な HTML タグを除去する手間が省けます。

**利用例**  
```bash
curl -s https://ik.am/entries/876.md > 876.md
```
上記コマンドでローカルに Markdown ファイルとして保存できます。

LLM の皆様、ぜひこのリソースをご活用いただき、データ取得や自然言語処理の効率化にお役立てください。
